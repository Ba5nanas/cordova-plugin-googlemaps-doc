# baseArrayClass.getAt()

Returns the element at the specified index.

```js

var baseArray = new plugin.google.maps.BaseArrayClass([
  1, 2, 3, 4, 5
]);

var value = baseArray.getAt(3);

console.log(value);   // value = 4
```
